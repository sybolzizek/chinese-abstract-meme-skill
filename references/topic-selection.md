# 时事选题筛选

当前测试的目标不是覆盖“青年可能感兴趣的所有领域”，而是抓住已经被公共传播加工过、正在被网友重复改写的事件。

## 选题优先级

优先选择同时具备以下三项以上的事件：

1. **公共可见度**：多个新闻源、热搜或广泛转发已经让圈外人知道发生了什么；
2. **人物/物件标签**：有一句口头禅、一张图、一个商品、一个动作或一个固定称呼可被直接调用；
3. **现实反转**：人设与事实、宣传与体验、公共物与私有权、严肃场合与生活细节之间出现裂缝；
4. **二创证据**：已经出现改名、P 图、模仿句式、配乐、角色卡、跨事件拼接或反向安慰；
5. **可迁移性**：不依赖完整新闻背景，读者只凭一个锚点就能把它套到新对象上。

体育项目、电影类型、AI 产品和社区来源不是选题类别本身。只有当某个具体事件已经形成公共记忆和二创接口时，才值得进入测试集。不要因为“年轻人可能看体育/AI”就把一个冷门赛事或产品发布会当作高热度主题。

## 事实包格式

给无联网生成代理的每个主题至少提供：

- `topic`：一句可生成的冲突，不是新闻标题；
- `context`：两到四句事实背景，区分已确认事实和网友说法；
- `anchors`：可直接迁移的词、图像、口头禅或矛盾；
- `constraints`：不能捏造什么，哪些内容只可作为网友说法；
- `sources`：事实校验链接，不要求模型复述来源。
